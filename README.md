# Python for Data Analysis - CMU Graduate Work

## Jupyter notebook portfolio from Carnegie Mellon University's graduate 'Python2' Course
## The curriculum applied primarily to data analysis and interacting with web APIs

*Week 1 & 2 - Data Analysis*
Security log and dataset analysis using pandas, numpy, and matplotlib.
Covers data cleaning, aggregation, and visualization of structured datasets.

*Week 3 - Automation with Loops and Functions*
Reusable functions design for parsing and processing data

*Week 4 - Web Scraping & HTML Parsing*
Automated data collection and parsing on public websites

## Main tools used include
Python 3 | Juptyer Notebooks | pandas | numpy | matplotlib | BeautifulSoup
